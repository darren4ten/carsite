清障车全名为道路清障车，又称拖车、道路救援车、拖拽车，具有起吊、拽拉和托举牵引等多项功能，清障车主要用于道路故障车辆，城市违章车辆及抢险救援等。清障车按类别主要分为：拖吊连体型、拖吊分离型，一拖一型，平板一拖二型，多功能清障车，液压自动夹紧型
