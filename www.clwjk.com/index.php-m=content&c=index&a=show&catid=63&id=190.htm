加油车是装运送油类物料的车辆，加油车上装备可随车移动的加油机，给受油设备，如车辆、农机、锅炉、施工机械及宾馆等单位配送燃料的车辆。加油车功能：自吸油料，自出油料，电脑税控加油，流动中加油等功能 。加油车（油车）主要用作石油的衍生品（汽油、柴油、原油、润滑油及煤焦油等油品）的运输和储藏。流动加油车相当于是一座移动的加油站，可随时随地给所需车辆进行供油。
