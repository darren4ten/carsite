冷藏车是指用来运输冷冻或保鲜的货物的封闭式厢式运输车，冷藏车是装有制冷机组的制冷装置和聚氨酯隔热厢的冷藏专用运输汽车，冷藏车可以按生产厂家、底盘承载能力、车厢型式来分类。冷藏车常用于运输冷冻食品（冷冻车），奶制品（奶品运输车）、蔬菜水果（鲜货运输车）、疫苗药品（疫苗运输车）等
