福田国三（5方）不上户吸粪车车型简介

MODEL INTRODUCTION

139-5777-0286 / 丽丽经理
整车型号 BJ5072GXE-G1 底盘型号 BJ1073VEJEA-A
发动机型号 锡柴88马力 容积 5方
外形尺寸 5995X1915X2150 排放标准 国三
轴距 3300 轮胎型号 7.0-16

吸粪车主要为城乡环卫部门抽吸和运送粪便及其它污水。适用于抽吸粪便，污水、污泥浆和混有较小悬浮杂物的液体，适用于市政环卫、大中小型工矿企业，小区、学校，化粪池清理转运，城市管道清理疏通，工厂沼液沼渣清理等。与洒水车，垃圾车，并称为三大环卫车型。长期以来受到国家相关扶持政策的帮助和关怀，该类车辆不需交纳购置税。
